📌 Pool Table Tracker

A lightweight and flexible WordPress plugin for managing pool table tracking, darts board management, and game activity monitoring inside WordPress.

🚀 Description

Pool Table Tracker helps you manage recreational game setups like pool tables and darts boards directly from your WordPress dashboard.

It provides simple tracking tools, admin management, and frontend shortcodes for displaying game interfaces.

Designed to be lightweight, fast, and easy to extend.

✨ Features
🎱 Pool table tracking system
🎯 Darts board management system
⚡ Lightweight and fast performance
🧩 Easy shortcode integration
🎛 Admin-friendly structure
🔌 Extendable architecture for future features
📦 Installation
Download or clone the plugin

Upload folder to:

/wp-content/plugins/pool-table-tracker/
Activate from WordPress Admin → Plugins
Use shortcodes anywhere on your site
🧩 Shortcodes
🎱 Pool Table Tracker
[pool_table_tracker]

Displays the pool table tracking interface including status, activity, or booking system (based on your implementation).

🎯 Darts Board Manager
[darts_board_manager]

Displays darts board tracking and management interface.

⚙️ Requirements
WordPress 5.8+
PHP 7.4+
MySQL 5.6+
